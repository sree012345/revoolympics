# Portrait Game Template

Reference resolution: **1080 × 1920**

## Purpose

Starting point for portrait Unity WebGL games. Not a commercial game.

## Will Include (Version 1+)

- Shared bridge integration
- Portrait orientation configuration
- Loading scene, game placeholder, result placeholder
- Localization foundation
- Game manifest
- WebGL build configuration
- Standard quit handling

## Usage

Clone or copy this template to `unity/games/{game_id}/` when creating a new portrait game.
