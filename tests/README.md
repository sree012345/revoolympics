# Cross-Platform Tests

Integration, end-to-end, security, and performance tests spanning multiple Revo Olympics components.

| Directory | Purpose |
|-----------|---------|
| `integration/` | Flutter + Firebase + Unity integration |
| `end_to_end/` | Full user journey tests |
| `security/` | Rules, auth, entitlement boundaries |
| `performance/` | Load and benchmark tests |
| `fixtures/` | Shared test data |

Implementation begins in feature sprints.
